# Fairness SDK: Анализ этичности и непредвзятости ИИ

Программный комплекс для квантификации метрик непредвзятости (Fairness) и оценки статистического смещения (Bias) в моделях машинного обучения. Система использует распределенную архитектуру очередей задач для масштабируемых вычислений.



## 1. Системные требования

| Компонент | Спецификация | Назначение |
| :--- | :--- | :--- |
| **Язык** | Python 3.10+ | Среда исполнения |
| **Брокер** | Redis / RabbitMQ | Маршрутизация задач |
| **Платформа** | Windows / Linux | Поддержка различных ОС |

## 2. Развертывание среды

Для изоляции зависимостей рекомендуется использовать виртуальное окружение.

```bash
# Создание и активация окружения (Windows)
python -m venv venv
venv\Scripts\activate

# Установка зависимостей
pip install -r requirements.txt

## 3. Аудит безопасности и контроль качества (DevSecOps)

# 1. Проверка зависимостей на известные уязвимости (CVE)
# Требует: pip install safety
safety check -r requirements.txt

# 2. Статический анализ безопасности (SAST)
# Требует: pip install bandit
bandit -r . -f custom

# 3. Линтинг (проверка стиля кода PEP 8)
# Требует: pip install flake8
flake8 . --count --select=E9,F63,F7,F82 --show-source --statistics

## 4. Запуск системы

# 4.1. Воркер Celery
celery -A tasks worker --pool=solo --loglevel=info

# 4.2. Основное ядро приложения
python main.py

# 4.3. Генератор задач
python run_tasks.py

# 4.4. Генерация метрологических артефактов (Визуализация)
python plot_training.py

python generate_plots.py

# 4.5. Проверка безопасности
python test_security.py

## 5. Структура проекта

main.py — Точка входа и глобальная конфигурация.

tasks.py — Определение асинхронных задач (вычисление метрик).

run_tasks.py — Скрипт пакетной обработки и инициации тестов.

requirements.txt — Спецификация библиотек.

.gitignore — Правила исключения временных файлов.
